# SIT305 Quiz App (Task 3.1C)

## Features
- Multiple choice quiz with submit + next
- Correct answer turns green
- Wrong selected answer turns red and correct turns green
- Progress bar updates per question
- Results screen shows final score
- Take New Quiz keeps user name
- Finish closes the app

## How to Run
1. Open in Android Studio
2. Sync Gradle
3. Run on emulator or device
